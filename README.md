## 🚀 Featured Projects

### [Codemia](https://github.com/Bitsnbytes14/Codemia) — Automated Coding Assignment Grader
- Scalable, production-ready grading platform for educational institutions
- Teachers create assignments with test cases; submissions auto-graded on correctness, runtime & memory
- **Multi-language support:** Python, JavaScript, C++, Java, C
- Built-in **plagiarism detection** and real-time analytics dashboards
- Horizontally scalable with **load-balanced, queue-based architecture**

**Tech:** `Node.js` `Docker` `Queue Architecture` `Load Balancing` `Multi-language Execution`

---

### [AI Resume Builder](https://github.com/Bitsnbytes14/AI-Resume-Builder) — ATS-Optimized Resume Platform
- Full-stack platform generating ATS-friendly resumes with AI-powered personalized suggestions
- Built the complete backend API, OpenAI integration pipeline, and MongoDB data layer from scratch

**Tech:** `React` `Node.js` `Express` `MongoDB` `OpenAI API`

---

### [Vairava — ACL Graft Predictor](https://github.com/Bitsnbytes14/Vairava---ACL-Graft-Predictor) — Medical ML Model
- ML model predicting ACL graft sizes from patient data using synthetic medical datasets
- Complete end-to-end pipeline: data generation → training → evaluation → visualization

**Tech:** `Python` `Scikit-learn` `Pandas` `Matplotlib`
